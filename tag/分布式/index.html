Spring Security 初识

Spring Security或者说web安全最核心的两个部分即：认证（Authentication） 和 授权（Authorization）。

  • 认证：用户是否能登录
  • 授权：用户是否有权限去做某些事情

即不用的用户登录之后有不同的权限可以去做某些事情。
